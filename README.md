# Angular Todo List Application

This is a simple Todo List application built with Angular, demonstrating core Angular concepts including components, services, routing, and forms.

## Features

- View list of tasks
- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Responsive design

## Project Structure

```
ng-training-assignment-1/
├─ src/
│  ├─ app/
│  │  ├─ components/
│  │  │  ├─ task-list/
│  │  │  └─ task-form/
│  │  ├─ models/
│  │  ├─ services/
│  │  └─ shared/
│  ├─ assets/
│  └─ styles/
```

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
ng serve
```

3. Open your browser and navigate to `http://localhost:4200`

## Development

- `ng serve` - Start development server
- `ng build` - Build the application
- `ng test` - Run unit tests
- `ng e2e` - Run end-to-end tests
